### GitHub で戻す手順（簡単）

1. 上の **コミット**（7 文字でも可）をコピーする。  
2. GitHub リポジトリ → **Commits** → 該当コミットを開く。  
3. **Browse files** で当時のファイルを確認するか、  
4. **Revert**（そのコミットの変更だけ打ち消す）や、ローカルで `git checkout <hash> -- path` でファイル単位復元。

機能追加をまとめて戻す場合は、そのコミットに対して **Revert** を 1 回押すのが安全です。
